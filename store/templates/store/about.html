About Project

This is an ECOMMERCE WEBSITE, with all the functionalities, such as User Cart and PayPal Payment processing.
This entire project is created with the PYTHON WEB FRAMEWORK - DJANGO.

Anybody can use this web-app. They can do so as a Guest User, or they can create an account from the backend.
This web-app also uses some concepts of javascript, which is used to send site cookies to the Python backend for processing.
For a good User Interface, I have used the Python Imaging library called 'Pillow'.
I have also used SQLite Database to store the User's orders and Cart Data for their future use and easy accessibility.
